Model Description

LDA is a topic modeling technique that uses a set of assumptions about how the writer generates the document. First, it assumes that the person is going to decide on the number of words according to a descrete distribution.

Second, a topic mixture is chose based on another distribution. Each word in the document is generated by first picking the topic and then selecting the word. LDA back tracks this process.
